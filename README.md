# AI-Powered Multi-Vendor E-Commerce System

A production-grade, AI-driven, multi-vendor marketplace with Flutter frontend, microservice backend, and ML-powered search and recommendations.

## Features
- Vendor onboarding & store management
- Customer shopping app
- Admin web dashboard
- AI smart search (NLP)
- Recommendations engine
- Image search
- Secure checkout & payments
- Real-time order tracking
- Multi-platform deployment

## Tech Stack
- Flutter (Customer & Vendor Apps)
- Node.js + NestJS (Backend APIs)
- Python + FastAPI (AI Services)
- PostgreSQL, Redis, Elasticsearch
- Docker, Kubernetes
- Firebase / S3 for media
- GitHub Actions CI/CD
